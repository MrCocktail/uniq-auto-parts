# Eclipse Starter for Jakarta EE
This is a sample application generated by the Eclipse Foundation Starter for Jakarta EE.

You can run the application by executing the following command from the directory where this file resides.
Please ensure you have installed a [Java SE implementation](https://adoptium.net) appropriate for your
Jakarta EE version and runtime choice (this sample assumes Java SE 17). The [Maven Wrapper](https://maven.apache.org/wrapper/)
is already included in the project, so a Maven install is not actually needed.

On Windows, use:

```
.\mvnw.cmd clean package wildfly:dev
```

On macOS/Linux, use:

```
./mvnw clean package wildfly:dev
```

```
./mvnw clean package wildfly:run
```

Once the runtime starts, you can access the project at [http://localhost:8080/uniq-auto-parts](http://localhost:8080/uniq-auto-parts).

The application creates a first employee account automatically on startup if none exists:

Username: `admin`
Password: `Admin123!`

The JPA connection targets MySQL on `localhost:3306` and the database `Db_AutoParts`.

By default, credentials are read from `persistence.xml`, but you can override them with environment variables:

- `APP_DB_DRIVER`
- `APP_DB_URL`
- `APP_DB_USER`
- `APP_DB_PASSWORD`

Example (PowerShell):

```powershell
$env:APP_DB_URL = "jdbc:mysql://localhost:3306/Db_AutoParts?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC"
$env:APP_DB_USER = "root"
$env:APP_DB_PASSWORD = "your_real_password"
.\mvnw.cmd clean package wildfly:dev
```
